期間：3ヶ月

### 自社開発CMSの運用保守

期間：2年以内

Movable typeをベースにした自社CMSの運用保守業務になります．私がジョインした際にはすでに運用フェーズだったため，主に営業経由で上がってくるお客様からの不具合分析，対応を担当しました．

お客様の不満が少しでも早くなくなるよう，こまめな連絡と迅速な調査，回答をするよう努めておりました．
